Introduction


This User Guide describes Shape2Earth Version 1.55 for MapWindow GIS Version 4.7.

Welcome to Shape2Earth for MapWindow GIS!  Shape2Earth is the easiest and cheapest way to convert GIS data into Keyhole Markup Language (KML) for viewing in Google Earth!

What Can I Do With Shape2Earth?

Shape2Earth for MapWindow GIS provides the following functionality:

  1. Convert Shapefiles to KML - Organize and prepare your shapefile for presentation in MapWindow GIS, and then export the data into Keyhole Markup Language (KML) for distributing and viewing in Google Earth and Google Maps.
  2. Apply Thematic Symbology to Your Data - Use  MapWindow GIS to thematically map your data using a 'Single Symbol', 'Unique Values', 'Equal Breaks', or a 'Continuous Ramp'. 
  3. Create Quick Google Earth Overlays - Convert your carographic map in MapWindow GIS into a image overlay for viewing in Google Earth. 
  4. Georeferenced Image Capture - Capture an image from Google Earth to view in MapWindow or another GIS.

New Features in Version 1.55

This version of Shape2Eath has a number of improvements and new features, such as:

  • Bug fixes for converting complex spatial geometries on non-English operating systems.
  • Sorting of data into folders named by a user selected data from the shapefile.
  • Improvements to the accuracy of georeferenced Google Earth imagery.
  • Ability to Georectify Google Earth Pro Images (paid version only).
    This capability only works with Google Earth versions prior to version 7.1.5.1557
    This functionality is being replaced by Image2Earth
  • User interface for adding Screen Overlays (paid version only).

Installing Shape2Earth

Install the following software.

  1. Download and install Google Earth
  2. Download and install Shape2Earth for MapWindow
