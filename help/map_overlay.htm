Exporting the MapWindow GIS View as a KML Image Overlay


You can export cartographically rendered maps created in MapWindow GIS into KML Image Overlays. This is very useful for showing large, complex, data effectively on Google Earth

Exporting Image Overlays

Add data to MapWindow GIS and apply symbols for each shapefile.  Zoom into the are on the map you would like to export, and then select 'Quick Map Overlay' from the Shape2Earth menu.  A Save dialog box will open to let you save the file on your hard drive.


The Exported overlay is automatically loaded into Google Earth
