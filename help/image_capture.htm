Capturing Geo-Rectified Images from Google Earth


NOTE: This functionality is being replaced by Image2Earth

Shape2Earth allows you to create a world file and projection file as part of the image copying process made available by Google Earth. Please refer to the Google Earth Terms of Service to ensure your use of this capability meets complies with any Google Earth restrictions (this may change over time).

Saving Georectified Images

Shape2Earth can capture black and white images in evaluation mode, and color images when licensed.  The captured image will be saved as a jpg, along with a .jpw world file, and .prj projection file  These images can then be loaded into MapWindow, or other GIS software 

Note:

This capability only works with Google Earth versions prior to version 7.1.5.1557.

This functionality is being replaced by Image2Earth

Images will be significantly more accurate if the Google Earth Terrain is turned off.

The World File and Projection file are written in WGS84.  The images can be reprojected using MapWindow, or other GIS software.

Georeference Google Earth Pro Image

Google Earth Pro allows you to save images at a very high resolution (up to 4,800 pixels).  You can apply a world file captured using the "Save Color Image" method by doing the following.

  1.  Zoom into the area on Google Earth that you would like to capture.
  2. Capture the current View as a georeferenced image and save to file.
  3. Without moving the View in Google Earth - Capture the same view using Google Earth Pro, and overwrite the image that you just captured (Ctrl + Alt + S).
  4. Click 'Georectify Google Earth Pro Images', and select the image file to rewrite the World File.

Reprojecting Image using the tools built into MapWindow

From the GIS Tools menu (If you do not see this menu, make sure it is activated under Plug-ins), select Image, and then Reproject Image.

Select the image from the open file dialog box.  You will see its information in the Selected Images dialog (below).  You can add more images here to be re-projected if they are all using the same system.


When you are ready, click OK , and the enter the new projection system into the Choose Projection dialog.


Once completed, an information box will come up.  Click the red x on the upper right to close.  Your reprojected files will be in the same directory as the original with the with "_Reprojected" appended to the file name
