Adding a Screen Overlay to Google Earth


Shape2Earth allows you to create add a custom graphic as a screen overlay, such as a business logo.

Add Screen Overlay

Select 'Advanced Tools' from the Shape2Earth menu:


 A Toolbar with six buttons will appear in the upper left hand corner of the Google Earth Globe Window.

The toolbar will include the following button

  • Add a Screen Overlay - Select an image and place it on the globe window to save as a Screen Overlay

To add your Screen Overlay:

  1. Select the Add Screen Overlay button
  2. Select the image you would like to add from the Open dialog
  3. Move and/or resize the image form to the location you would like it on the Globe Window
  4. Double-Click anywhere on the form to set the location of your image
Note:

Once the Screen Overlay has been added, you will not be able to edit it using the tool. To remove the Screen Overlay, you can select "Discard" when you close Google Earth, or you can select "Save" and then delete it from "My Places" the next time you open Google Earth.

The Screen Overlay is not currently added to your exported shapefile KML. You can copy and paste the Screen Overlay into any KML, and then resave that KML.
